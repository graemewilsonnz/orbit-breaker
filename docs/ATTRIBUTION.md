# Orbit Breaker Attribution

## Runtime content

Orbit Breaker does not ship externally sourced art, fonts, music, sound samples, narrative text, or
gameplay data. Canvas visuals, particle effects, and Web Audio cues are original and generated by
the project code.

## Development tooling

The development and verification toolchain uses TypeScript, Vite, Vitest, Playwright, ESLint, and
Prettier. These packages are development dependencies only and are not bundled as runtime content.
Exact versions and transitive packages are recorded in `package-lock.json`; their own licence terms
remain with their respective authors.
